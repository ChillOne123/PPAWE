## 🌟 致各位学弟学妹

欢迎来到计算社会科学（CSS）的效率世界。

在**PA (Public Administration)**的期末季，或是各类学术竞赛的备赛期，动辄几万字的论文压下来，时间紧任务重。很多时候，我们把大量的精力耗费在了知网查重、机械地复制粘贴摘要、以及痛苦地手动核对 `(作者, 年份)` 这种低效的体力劳动上。

之前在做公共部门绩效管理系统的相关研究时，我曾和团队耗费了巨大的精力去人工编码政策文本。那种纯手工的“暴力打磨”虽然基础，但也让我深刻意识到：**与其在传统的纯理论框架里做空洞的推演，不如掌握一些“硬核”的技术手段，用工具杠杆来重塑我们的学术生产力。** 

这个轻量级的 RAG（检索增强生成）工具，就是为此诞生的。它不是一个帮你“盲目水稿”的代写外挂，而是一个**受控的文献插桩校对官**。通过解析结构化的元数据，它能帮你快速提炼核心变量、梳理文献综述，并在正文中精准打上引文标签。

希望这个小工具能帮你们从繁杂的工程细节中抽身，把宝贵的时间留给**研究方法的设计**和**核心逻辑的推敲**。

## 🛠️ 核心功能

- **Excel 元数据驱动**：告别极其难用的 PDF 双栏解析，直接读取知网导出的结构化 Excel（包含作者、年份、标题、摘要），数据干净，极大幅度降低模型幻觉。
- **动态知识库注入**：将你的文献库转化为大模型的上下文，确保每一句论述都有迹可循。
- **自动引文插桩**：根据你输入的论文草稿或撰写指令，模型会在生成的学术文本句末自动匹配并插入标准格式的 `(作者, 年份)`。

## ⚙️ 极速上手指南

### 1. 环境准备
确保你安装了 Python 3.8 及以上版本。在终端中运行以下命令安装依赖：
```bash
pip install -r requirements.txt

```

### 2. 准备文献数据

1. 登录中国知网 (CNKI)，搜索你的研究主题。
2. 勾选相关的核心期刊文献（建议 30-50 篇）。
3. 选择 **“导出/参考文献”** -> **“自定义格式”** 或者直接导出为 **Excel** 格式。
4. **⚠️ 重要**：请打开下载好的 Excel，确保表格的表头包含这四列（如果没有请手动重命名）：`作者`、`年份`、`标题`、`摘要`。

### 3. 获取 API Key

本项目默认使用兼容 OpenAI 接口的 API（代码中预设为硅基流动调用的 DeepSeek 模型）。

* 请前往对应平台注册并获取你的 API Key。

### 4. 一键启动

在项目根目录下运行：

```bash
streamlit run app.py

```

浏览器会自动打开 `http://localhost:8501`。在侧边栏填入 API Key 并上传你的 Excel 文件，即可开始享受高效率的学术撰写！

## ⚠️ 学术诚信与风险提示 (Disclaimers)

1. **防范幻觉**：大模型偶尔仍会有“微弱幻觉”。请务必抽查引用的文献是否真的提出了对应的观点。
2. **工具定位**：此工具旨在**辅助文献综述的梳理**和**引文格式的规范化**，严禁使用本工具进行全自动的无脑生成。优异的论文永远源自你扎实的逻辑与深刻的公共管理洞察。
3. **数据隐私**：知网数据带有版权属性，`.gitignore` 中已默认忽略 `.xlsx` 和 `.csv` 文件，请勿将批量下载的文献集 Push 到公开仓库。
